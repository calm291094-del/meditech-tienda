💊 MediTech 💻 - Tienda de Medicamentos y Tecnología

Tu tienda confiable para medicamentos y hardware de última generación 🚀

📖 Descripción

MediTech es una tienda online moderna y completamente funcional que combina dos mundos: **medicamentos** y **tecnología**. Desarrollada como una aplicación web estática con GitHub Pages, ofrece una experiencia de usuario profesional con gestión de productos, sistema de autenticación, carrito de compras y panel de administración.

🎯 Características Principales

✨ **Para Clientes:**
- 🛒 Carrito de compras intuitivo
- 🔐 Sistema de registro e inicio de sesión
- 📱 Diseño 100% responsive (PC, tablet, móvil)
- 🔍 Búsqueda y filtrado de productos
- 📧 Envío de pedidos por correo electrónico
- 💬 Chatbot con IA para consultas

🔒 **Para Administradores:**
- 📊 Panel de administración completo
- 🖼️ Gestión de productos (CRUD)
- 📸 Subida de imágenes desde PC/móvil
- 🎨 Editor de carrusel y textos en vivo
- 👥 Gestión de usuarios registrados
- 📈 Estadísticas y métricas en tiempo real
- 📝 Historial de vistas y pedidos
- 💾 Sincronización con GitHub API

🎯 Funcionalidades Detalladas
🛒 Sistema de Compras

    Explorar productos: Navega por categorías o usa la búsqueda
    Agregar al carrito: Haz clic en el botón "+" en cualquier producto
    Ver carrito: Accede al ícono del carrito (solo usuarios registrados)
    Solicitar pedido: Envía tu pedido por correo con un clic

🔐 Sistema de Autenticación

    Registro: Crea una cuenta con nombre, usuario, email y contraseña
    Login: Inicia sesión con tus credenciales
    Sesión persistente: Tu sesión se mantiene al recargar la página
    Roles: Admin (acceso total) y Cliente (compras)

🎨 Panel de Administración
Accede con la cuenta root para:

    ✅ Productos: Agregar, editar, eliminar, cambiar estado
    🖼️ Carrusel: Editar imágenes y textos del slider principal
    ✏️ Textos: Modificar títulos y descripciones de la página
    👥 Usuarios: Ver todos los usuarios registrados
    📊 Estadísticas: Métricas de ventas, vistas, ingresos
    📝 Historial: Registro de todas las visitas
    📦 Pedidos: Ver todos los pedidos recibidos

💬 Chatbot con IA

    Respuestas inteligentes sobre medicamentos y tecnología
    Información de productos en tiempo real
    Asistencia 24/7
    Powered by Pollinations AI (gratis, sin API key)

🔒 Seguridad
⚠️ Importante
Este proyecto usa GitHub API como backend. Consideraciones de seguridad:
✅ Buenas prácticas implementadas:

    Tokens con permisos mínimos
    Validación de inputs
    Sanitización de datos
    Roles de usuario (admin/cliente)

⚠️ Limitaciones:

    El token es visible en el código fuente (frontend)
    No usar para datos sensibles reales
    Solo para demostración o proyectos personales

🔐 Recomendaciones

    Nunca compartas tu token públicamente
    Usa tokens con expiración (30-90 días)
    Crea un token específico solo para este proyecto
    No almacenes datos sensibles en los JSON
